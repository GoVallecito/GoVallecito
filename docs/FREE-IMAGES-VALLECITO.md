## Attribution plan (required for Tier 1)
- Each placed image: short visible credit (small caption or corner overlay) — e.g. "📷 Jeffrey Beall ·
  CC BY 4.0" — OR a `#photo-credits` section on sources.html listing every image, author, license, and
  link, with each image's alt/figure linking there. The credits-section approach is cleaner; CC BY allows
  it ("reasonable to the medium").
- CC BY-**SA** images (Lund aerial, LaRocco peaks, Emerald Lake, CDT): display/crop fine, but don't bake
  into composite branded graphics (ShareAlike obligations). Plain placement on a page is fine.
- Keep this file updated as the rights record: which image, where used, license, source URL.
